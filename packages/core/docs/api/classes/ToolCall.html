    Class ToolCall

    Represents a tool call request from the model

    This is returned by the model when it wants to use a tool.

    Example

    const call = new ToolCall({
      id: 'call_abc',
      name: 'calculator',
      arguments: { x: 5, y: 3, op: 'add' },
      providerFormat: ToolCallFormat.OPENAI
    });
