    Class RetryManager

    Retry Manager

    Handles automatic retry logic with exponential backoff for provider calls. Classifies errors and applies appropriate retry strategies.

    Example

    const manager = new RetryManager();

    // Execute with automatic retry
    const result = await manager.executeWithRetry(async () => {
      const response = await fetch('https://api.example.com/data');
      return response.json();
    });

    // Check metrics
    console.log(manager.getMetricsSummary());
    // { totalAttempts: 2, successful: 1, failed: 1, successRate: "50.0%", ... }
